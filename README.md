# 페어 매칭 관리

## 기능
- 기능 선택 안내문을 출력한다.
- 기능을 선택한다.
  - [예외] 공백을 입력한 경우 
  - [예외] 1, 2, 3, Q가 아닌 다른 값을 입력한 경우
- 페어 매칭 선택 시 과정, 레벨, 미션 정보를 출력한다.
- 과정, 레벨, 미션을 입력받는다.
  - 매칭 정보가 존재할 경우 다시 매칭할 지 물어본다.
  - 아니오를 입력 시 다시 과정, 레벨, 미션을 입력받는다.
  - [예외] 공백을 입력한 경우
  - [예외] 하나의 정보라도 누락된 경우
  - [예외] 존재하지 않는 과정, 레벨, 미션을 입력한 경우
- 크루들의 이름을 파일 입출력을 통해 불러온다.
- 크루들의 이름 목록을 List<String> 형태로 준비한다.
- 페어를 두명씩 매칭한다.
  - 크루 목록의 순서를 랜덤으로 섞는다.
  - 앞에서부터 순서대로 두명씩 페어를 맺는다.
  - 페어 매칭 대상이 홀수인 경우 한 페어는 3인으로 구성한다.
  - 같은 레벨에서 이미 페어를 맺은 크루와는 다시 페어로 매칭될 수 없다.
  - 페어를 맺었던 크루가 다시 매칭된다면 크루 목록의 순서를 다시 랜덤으로 섞어서 매칭을 시도한다.
  - 3회 시도까지 매칭이 되지 않거나 매칭을 할 수 있는 경우의 수가 없으면 에러 메시지를 출력한다.

- 페어 매칭 결과를 출력한다.
- 페어 조회 기능을 출력한다.
  - [예외] 매칭된 페어가 존재하지 않는 경우
- 페어 초기화 시 모든 페어 정보가 초기화된다.
